# Nestjs CRUD

## Description

Simple Nestjs CRUD application

## Installation

```bash
$ npm install
```

## Running the app

Start docker postres container

```bash
npm run start:dev:db
```
Start the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
